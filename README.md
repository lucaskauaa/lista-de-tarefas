# ✅ Lista de Tarefas

Site de uma lista de tarefas simples, feito com JavaScript, HTML e CSS.

## ⚙️ Funcionalidades

1. Adicionar tarefas a lista
2. Marcar como feito
3. Excluir tarefa
4. Troca de temas 
5. Responsividade

## ▶️ Como Executar

🔗 [Clique aqui para acessar o site.](https://lucaskauaa.github.io/lista-de-tarefas/html/index.html)
